What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment, a core practice in modern cloud computing. Continuous Integration automates the process of merging and testing code changes to ensure stability. Continuous Deployment then automatically delivers these updates to the hosting service. This approach reduces manual errors, speeds up development, and ensures the live site is always up to date.

My Pipeline

After I push my code to GitHub, a workflow is automatically triggered. It runs on an Ubuntu machine to check if all files exist, and then deploys the site to GitHub Pages.
